# GoToKart documentation

Static documentation site for [github.com/gotokart](https://github.com/gotokart), built with [Next.js](https://nextjs.org/) + [Nextra](https://nextra.site/) (same stack style as [manishdotkr/docs](https://github.com/manishdotkr/docs), with **custom pages** tailored to GoToKart).

## Features

- **Docs UI:** Nextra docs theme, dark mode, gold gradient branding aligned with GoToKart.
- **Commit activity:** The **Commit activity** page shows a timeline of commits to **this repo**, grouped by day. Data is generated by `scripts/generate-activity.mjs` from `git log` on every `npm run dev` / `npm run build`.
- **CI/CD:** GitHub Actions builds a static export (`out/`) and deploys to **GitHub Pages** (see `.github/workflows/deploy-docs.yml`).

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
```

Static files are written to `out/`.

## GitHub Pages setup

1. In the **docs** repository on GitHub: **Settings → Pages → Build and deployment → Source: GitHub Actions**.
2. The workflow sets `BASE_PATH=/docs` so the site works at **`https://gotokart.github.io/docs/`** (repository name `docs` under the `gotokart` organization).
3. If your repository name or Pages URL differs, change the `BASE_PATH` env in `.github/workflows/deploy-docs.yml` and update the `homepage` field in `package.json` to match.

## Repository layout

| Path | Purpose |
|------|---------|
| `pages/` | MDX documentation (not copied from external projects) |
| `components/ActivityTimeline.tsx` | Commit timeline UI |
| `scripts/generate-activity.mjs` | Writes `generated/activity.json` from git history |
| `theme.config.tsx` | Logo, footer, theme colors |
