## High-level architecture

<p className="gk-arch-note">
  Single source architecture diagram (runtime + delivery flow):
</p>

```mermaid
flowchart TB
  U[Users] --> FE[Frontend UI<br/>GitHub Pages]
  FE -->|JWT Auth + REST| BE[Backend API<br/>Spring Boot]
  BE -->|JPA/Hibernate| DB[(MySQL 8)]
  BE -->|BCrypt + JJWT| Auth[Spring Security<br/>JWT Filter]

  Dev[Developer] --> RepoFE[github.com/gotokart/frontend]
  Dev --> RepoBE[github.com/gotokart/backend]
  Dev --> RepoDocs[github.com/gotokart/docs]
  Dev --> RepoInfra[github.com/gotokart/infra]

  RepoFE --> CI[GitHub Actions / CI-CD]
  RepoDocs --> CI
  RepoInfra --> CI
  RepoBE --> EC2Deploy[SSM Send-Command<br/>EC2 Deploy]

  CI --> FE
  CI --> Docs[Docs Site<br/>GitHub Pages]
  FE --> DomainFE[gotokart.github.io/frontend]
  Docs --> DomainDocs[gotokart.github.io/docs]
  EC2Deploy --> DomainAPI[EC2 / localhost:8080]
```
