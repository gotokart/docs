## High-level architecture

<div className="gk-arch-note">
  Request flow: Browser loads storefront, storefront calls backend APIs, backend persists data in MySQL.
</div>

```mermaid
flowchart LR
  Browser[Browser] --> Pages[Frontend - GitHub Pages]
  Pages --> API[Backend API]
  API --> DB[(MySQL)]
  Dev[Git push] --> Actions[GitHub Actions]
  Actions --> Pages
```
