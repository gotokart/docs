# Frontend

The storefront is a **single-page** experience built with **HTML, CSS, and JavaScript** (no framework). It talks to the backend REST API.

## Files

- `index.html` — layout and sections (hero, auth, shop, cart, orders)
- `style.css` — dark luxury theme (gold accents, responsive layout)
- `script.js` — API calls, UI state, and rendering

## API base URL

The client uses a single constant for the API root (typically `http://localhost:8080/api` in development). Change it in `script.js` to point at your deployed API.

## Features (overview)

- Register / login / logout
- Product listing with search, filters, and sorting
- Cart with quantity and totals
- Place order and view order history

For details and customization, see the [frontend README](https://github.com/gotokart/frontend/blob/main/README.md).
