## Run locally

### Option A — Docker (recommended)

Requires Docker Desktop installed.

```bash
cd infra
docker compose -f docker-compose.local.yml up --build
```

This starts MariaDB, Spring Boot backend, and Nginx. Open `http://localhost`.

### Option B — Backend directly (no Docker)

Prerequisites: Java 21, Maven 3.9+, MySQL or MariaDB running locally.

**1. Create the database:**
```sql
CREATE DATABASE gotokart;
```

**2. Configure `backend/src/main/resources/application.properties`:**
```properties
spring.datasource.url=jdbc:mysql://localhost:3306/gotokart?useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=UTC
spring.datasource.username=root
spring.datasource.password=Root@1234

jwt.secret=GoToKartSuperSecretJwtKey2026!!VeryLongAndSecure@#$%
jwt.expiry-ms=86400000
```

**3. Start the backend:**
```bash
cd backend
mvn clean spring-boot:run
```

**4. Open the frontend:**

Open `frontend/index.html` directly in your browser. It auto-detects local mode and calls `http://localhost:8080/api`.

## How authentication works

1. Click **Login** → enter email and password
2. Frontend calls `POST /api/auth/login` → receives JWT token
3. Token saved to `localStorage`, sent as `Authorization: Bearer <token>` on admin requests
4. Session restored automatically on page reload
