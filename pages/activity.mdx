# Commit activity

This page lists **commits to this documentation repository**, grouped by **calendar day**. It also provides:

- a direct **See full commit history** link to GitHub
- an expandable **full commit list** inside docs for quick scanning

It is generated automatically from `git log` when you run `npm run dev` or `npm run build`, so it always reflects the history available in your checkout (CI uses a full clone so the timeline is complete).

<ActivityTimeline />

### How it works

1. `scripts/generate-activity.mjs` runs before dev and build.
2. It writes `generated/activity.json`.
3. The timeline below reads that file and links each commit to GitHub.
